# Code of Conduct

This project follows the Contributor Covenant v2.1. By participating, you agree to abide by its terms and to treat other contributors with respect.

Be respectful, avoid harassment, and act in good faith. Violations may result in removal from the project as described in the full Contributor Covenant.

For the full text, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
